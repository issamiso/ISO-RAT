# ISO-RAT
## setup Termux
```
pkg install git -y
git clone https://github.com/issamiso/ISO-RAT
cd ISO-RAT
chmod +x *
bash setup.sh 
```

``` 
apt install git -y
git clone https://github.com/issamiso/ISO-RAT
cd ISO-RAT
chmod +x *
bash kali.sh 
```
